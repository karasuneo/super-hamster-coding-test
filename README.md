## 実行方法

### サーバーを起動

```bash
# モジュールのインストール
npm install

# 開発者サーバーの起動
npm run dev
```

http://localhost:5173/ にアクセスする

## フォーマッターの実行方法

`prettier --write src && eslint --fix src`を実行します

```bash
# フォーマッターの実行
npm run format
```
